The Game Module¶

Game¶

class game.Game(displayclass, inputclass)¶

Game object:

  • map => game.Map
  • input => headless.Input or modpygame.Input
  • display => graphics.Display

contains the main loop

loop()¶
the main loop of bzflag. checks events, updates positions, and draws to the screen until the pygame window is closed.
update()¶
updates the game world
update_sprites()¶
adds and removes sprites from the display

Map¶

class game.Map(game)¶

manages the map data. populates the current map with bases, obstacles, teams and tanks.

handle_collisions(dt)¶

Handles the collision detecting process for a given object.

The obj parameter is the object, while the dt is the time elapsed since collisions were last handled for this object. As a result of calling this method, the object’s new position and status (alive, dead, etc.) are set.

shots()¶
iterate through all shots on the map
tanks()¶
iterate through all tanks on the map
update(dt)¶
update the teams
